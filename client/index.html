About the project

This project was developed by Fabrício Fernandes Ziliotti and Marcos Victor de Aquino Barra. It is the final project of the discipline Object Oriented Programming 2 of the Federal University of Uberlândia in 2018.

The main focus was the use of design patterns in its development.

HTML5, CSS3, SASS, ECMAScript 6, ChartJS and GULP technologies were used, as well as the use of Bootstrap 4 to facilitate the front-end.
